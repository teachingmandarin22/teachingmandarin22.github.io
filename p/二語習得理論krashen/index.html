二語習得理論：Krashen

Jul 01, 2023
1 minute read

二語習得的學說林林總總，總是不能繞開 Krashen，今天看了 Youtube 上這位 Steve Kaufmann 爺爺講得蠻有趣的。

當我們在學習語言時，中間會經過什麼過程呢？

Krashen 有五個假說來說明這個過程。

1.The Acquistion/Learning Hypothesis：

語言學習有兩種不同模式，一種是潛意識下自然而然「習得」的，另一種則是在正式課堂上有意識學習各種文法規則而「學習」到的。

2.The Natural Order Hypothesis：

基於大量語言學家的研究，Krashen 人們習得語言有一個共同的順序，而且跟難易度沒有關係（例如：第三人稱複數要加 s ，雖然簡單，但是比較晚學到）這邊並沒有解釋得非常清楚，聽起來像是兒童在學習語言時，語音、詞彙、語法的進展可能會有類似的進程？

3.The Monitor Hypothesis：

監控假說。說話者在說二語時同時檢查自己的語言合不合乎文法規則。這也就是為什麼 Krashen 認為學習文法規則不能幫助學生，因為一方面必須要召喚學習到的文法規則並應用、偵錯，另一方面還要想自己要說什麼，實在無法自然而然地表達。

4.The Input Hypothesis：

(近年來他修正他的假說為 Comprehensive Hypothesis了)：有意義的輸入很重要，就像「熟讀唐詩三百首，不會做詩也會吟」，好的 input 才能夠幫助學習，然而這個材料的難度要比學習者現在的程度再難一點點，也就是 i+1 的假說；Krashan 認為聽跟讀才是語言學習的重心，說跟寫則不是。

→ 因此，教師必須時常思考自己需要講什麼樣的語言？採用什麼樣的教學方法和技巧？

💡 → 「有意義的輸入」假說在學界影響深遠，不過近日也被學者們提出反省：語言輸入也是重要的。

5.The Affective Filter Hypothesis：

爺爺沒特別講到，這個假說認為，語言習得過程跟年齡沒有關係，不會在青春期就消失或受到損害了，成年人習得效果不彰，是因為情感因素阻撓，例如學習動機、焦慮或者自信心等等。

反思：

Krashen 的看法跟我現在所運用的華語教學有幾點不一樣：

老師應該精講多練，基本上老師越少開口越好，讓學生輸出>輸入是最重要的；老師越少開口，但是開口講解時，更需要講究技巧，要讓學生盡可能在最短時間就理解語法點的結構、用法、含義。

老師還是會講文法，但不是在講生硬的語言學理論，需要在乎文法自易至難的排列，但是不分析也不給太多的語法結構，而是找生活上的情境切入，並且重視操練以及鞏固，最後再設計提問，讓學生能夠學生自由表達的時間。

我開始覺得，不同的內容需要使用不同的教學方法，也需要考量學生年齡、程度、動機等等，無法一招打遍天下。
